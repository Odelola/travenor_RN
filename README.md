#### Repository

## HOW TO RUN

### Install the node_modules by running
```
yarn
```
### or
```
npm i
```
### To get the exact dependencies used
```
yarn -E
```
### or
```
npm i --save-dev
```
### And then run
```
npx expo start
```



### Technologies

- React Native
- Typescript
- Nativewind
- React Native Navigation and Bottom Tab Navigation
